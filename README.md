## Quick start

### Development

1. Clone Repo
   ```bash
   $ git clone https://github.com/diogofalken/csgo-binds.git
   ```
2. Change Directory
   ```bash
   $ cd csgo-binds/
   ```
3. Clean project
   ```bash
   $ npm run clean
   ```
4. Install dependencies

   ```bash
   $ npm install
   ```

5. Run development app
   ```bash
   $ npm start
   ```
